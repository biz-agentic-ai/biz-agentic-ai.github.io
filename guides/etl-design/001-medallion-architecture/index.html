лҚ°мқҙн„° л ҲмқҙнҒ¬к°Җ лҠӘмқҙ лҗҳлҠ” кіјм •#

лҚ°мқҙн„° л ҲмқҙнҒ¬м—җ нҢҢмқјмқ„ мҸҹм•„лҶ“кі  л°”лЎң л¶„м„қн•ҳл ӨлҠ” нҢҖмқҙ мһҲлӢӨ. мІҳмқҢм—” л№ лҘҙлӢӨ. CSV мҳ¬лҰ¬кі  SQL н•ң мӨ„мқҙл©ҙ кІ°кіјк°Җ лӮҳмҳЁлӢӨ.

3к°ңмӣ” м§ҖлӮҳл©ҙ мғҒнҷ©мқҙ лӢ¬лқјм§„лӢӨ. лҲ„к°Җ мҳ¬лҰ° нҢҢмқјмқём§Җ лӘЁлҘёлӢӨ. мӣҗліёмқём§Җ м •м ңлҗң кұҙм§Җ кө¬л¶„мқҙ м•Ҳ лҗңлӢӨ. к°ҷмқҖ л§Өм¶ң н…Ңмқҙлё”мқёлҚ° л¶Җм„ңл§ҲлӢӨ мҲ«мһҗк°Җ лӢӨлҘҙлӢӨ. лҚ°мқҙн„° лҠӘ(data swamp)мқҙлқјкі  л¶ҖлҘҙлҠ” мғҒнғңлӢӨ.

мӣҗмқёмқҖ лӢЁмҲңн•ҳлӢӨ. мӣҗліёкіј к°Җкіөл¬јмқҙ к°ҷмқҖ кіөк°„м—җ м„һм—¬ мһҲкё° л•Ңл¬ёмқҙлӢӨ. л Ҳмқҙм–ҙлҘј лӮҳлҲ„л©ҙ мқҙ л¬ём ңк°Җ н’ҖлҰ°лӢӨ.

Bronze, Silver, Gold#

л©”лӢ¬лҰ¬мҳЁ м•„нӮӨн…ҚмІҳлҠ” лҚ°мқҙн„°лҘј м„ё к°ңмқҳ л Ҳмқҙм–ҙлЎң лӮҳлҲҲлӢӨ. Databricksк°Җ мқҙлҰ„мқ„ л¶ҷм—¬м„ң л„җлҰ¬ нҚјмЎҢм§Җл§Ң, к°ңл…җ мһҗмІҙлҠ” м „нҶө DWмқҳ л Ҳмқҙм–ҙл“ң м ‘к·јкіј к°ҷлӢӨ.

мҶҢмҠӨ мӢңмҠӨн…ң вҶ’ [Bronze] вҶ’ [Silver] вҶ’ [Gold] вҶ’ BI / л¶„м„қ
               мӣҗліё м Ғмһ¬    м •м ңВ·н‘ңмӨҖнҷ”   л№„мҰҲлӢҲмҠӨ м§‘кі„

Bronze лҠ” мӣҗліё к·ёлҢҖлЎңлӢӨ. мҶҢмҠӨ мӢңмҠӨн…ңм—җм„ң к°Җм ёмҳЁ лҚ°мқҙн„°лҘј ліҖнҷҳ м—Ҷмқҙ м ҖмһҘн•ңлӢӨ. CSVл“  JSONмқҙл“  API мқ‘лӢөмқҙл“  мһҲлҠ” к·ёлҢҖлЎң. лҚ°мқҙн„° кі„ліҙ(lineage)мқҳ м¶ңл°ңм җмқҙлӢӨ. м—¬кё°м„ң лӯ”к°ҖлҘј л°”кҫёл©ҙ мӣҗліёмқ„ мһғлҠ”лӢӨ.

Silver лҠ” м •м ңмҷҖ н‘ңмӨҖнҷ”лӢӨ. Bronze лҚ°мқҙн„°мқҳ нғҖмһ…мқ„ л§һм¶”кі , мӨ‘ліөмқ„ м ңкұ°н•ҳкі , нӮӨлҘј нҶөн•©н•ңлӢӨ. “л¶„м„қм—җ м“ё мҲҳ мһҲлҠ” мғҒнғң"лЎң л§Ңл“ңлҠ” л Ҳмқҙм–ҙлӢӨ. л№„мҰҲлӢҲмҠӨ лЎңм§ҒмқҖ м•„м§Ғ л„Јм§Җ м•ҠлҠ”лӢӨ.

Gold лҠ” л№„мҰҲлӢҲмҠӨ кҙҖм җмқҳ м§‘кі„лӢӨ. нҢ©нҠё н…Ңмқҙлё”, м°Ёмӣҗ н…Ңмқҙлё”, KPI л§ҲнҠё. мөңмў… мӮ¬мҡ©мһҗк°Җ м§Ғм ‘ мҝјлҰ¬н•ҳлҠ” л Ҳмқҙм–ҙлӢӨ. DW лӘЁлҚёл§Ғ 1нҺё м—җм„ң лӢӨлӨҳлҚҳ мҠӨнғҖмҠӨнӮӨл§Ҳк°Җ м—¬кё° н•ҙлӢ№н•ңлӢӨ.

к°Ғ л Ҳмқҙм–ҙмқҳ м—ӯн• мқҙ лӘ…нҷ•н•ҳлӢӨлҠ” кІҢ н•өмӢ¬мқҙлӢӨ. Bronzeм—җм„ңлҠ” ліҖнҷҳн•ҳм§Җ м•ҠлҠ”лӢӨ. Silverм—җм„ңлҠ” л№„мҰҲлӢҲмҠӨ лЎңм§Ғмқ„ л„Јм§Җ м•ҠлҠ”лӢӨ. Goldм—җм„ңл§Ң л№„мҰҲлӢҲмҠӨ кҙҖм җмқҳ к°Җкіөмқҙ л“Өм–ҙк°„лӢӨ. мқҙ к·ңм№ҷмқҙ к№Ём§Җл©ҙ л Ҳмқҙм–ҙлҘј лӮҳлҲҲ мқҳлҜёк°Җ м—ҶлӢӨ.

м „нҶө DW л Ҳмқҙм–ҙмҷҖмқҳ лҢҖмқ‘#

DW лӘЁлҚёл§Ғ мӢңлҰ¬мҰҲ м—җм„ң Raw вҶ’ Staging вҶ’ Integration вҶ’ Mart кө¬мЎ°лҘј лӢӨлӨҳлӢӨ. л©”лӢ¬лҰ¬мҳЁкіј мқҙлҰ„л§Ң лӢӨлҘҙкі  м—ӯн• мқҖ кұ°мқҳ к°ҷлӢӨ.

л©”лӢ¬лҰ¬мҳЁм „нҶө DWн•ҳлҠ” мқј
BronzeRaw / Stagingмӣҗліё м Ғмһ¬, ліҖнҷҳ м—ҶмқҢ
SilverIntegration (3NF / Data Vault)м •м ң, н‘ңмӨҖнҷ”, нӮӨ нҶөн•©
GoldMart (Star Schema)л№„мҰҲлӢҲмҠӨ м§‘кі„, л¶„м„қмҡ©

м „нҶө DWм—җм„ңлҠ” Stagingкіј Integration мӮ¬мқҙм—җ ETL м„ңлІ„к°Җ л¬ҙкұ°мҡҙ ліҖнҷҳмқ„ мІҳлҰ¬н–ҲлӢӨ. л©”лӢ¬лҰ¬мҳЁмқҖ ELT нҢЁлҹ¬лӢӨмһ„мқҙлӢӨ. мқјлӢЁ Bronzeм—җ м Ғмһ¬н•ҳкі , DW м—”м§„ м•Ҳм—җм„ң SilverмҷҖ GoldлҘј л§Ңл“ лӢӨ. ліҖнҷҳмқ„ лі„лҸ„ м„ңлІ„к°Җ м•„лӢҲлқј DW м—”м§„мқҳ м»ҙн“ЁнҢ… нҢҢмӣҢлЎң мІҳлҰ¬н•ңлӢӨлҠ” м җмқҙ лӢӨлҘҙлӢӨ.

мқҙ мӢңлҰ¬мҰҲмқҳ мӢӨмҠө нҷҳкІҪ#

мӢңлҰ¬мҰҲ м „мІҙм—җм„ң мӮ¬мҡ©н•  лҸ„кө¬лҠ” м„ё к°Җм§ҖлӢӨ. м „л¶Җ л¬ҙлЈҢмқҙкі , Google Colabм—җм„ң нҒҙлқјмҡ°л“ң кі„м • м—Ҷмқҙ л°”лЎң лҸҢлҰҙ мҲҳ мһҲлӢӨ.

лҸ„кө¬м—ӯн• 
DuckDBлЎңм»¬ DW м—”м§„. Columnar Storage кё°л°ҳмқҙлқј BigQuery/SnowflakeмҷҖ к°ҷмқҖ л°©мӢқмңјлЎң лҸҷмһ‘н•ңлӢӨ
dbt-core + dbt-duckdbліҖнҷҳ л Ҳмқҙм–ҙ. SQLлЎң Bronze вҶ’ Silver вҶ’ GoldлҘј м •мқҳн•ңлӢӨ
Soda CoreлҚ°мқҙн„° н’Ҳм§Ҳ кІҖмҰқ. л Ҳмқҙм–ҙ к°„ н’Ҳм§Ҳ кІҢмқҙнҠёлҘј кұҙлӢӨ

DuckDBлҘј кі лҘё мқҙмң к°Җ мһҲлӢӨ. м„Өм№ҳк°Җ pip install н•ң мӨ„мқҙл©ҙм„ң, мӢӨм ң нҒҙлқјмҡ°л“ң DWмҷҖ лҸҷмһ‘ л°©мӢқмқҙ к°ҷлӢӨ. Parquet, CSVлҘј л„ӨмқҙнӢ°лёҢлЎң мқҪкі , SQLлЎң л¶„м„қн•ҳкі , Columnar Storageлқј м»¬лҹј кё°л°ҳ мҠӨмә”мқҙ лҗңлӢӨ. лЎңм»¬м—җм„ң лҸҢлҰ¬м§Җл§Ң нҒҙлқјмҡ°л“ң DWмқҳ м¶•мҶҢнҢҗмқҙлқјкі  ліҙл©ҙ лҗңлӢӨ.

Colab м…Җм—җм„ң м•„лһҳлҘј мӢӨн–үн•ҳл©ҙ мӨҖл№„ лҒқмқҙлӢӨ.

мғҳн”Ң лҚ°мқҙн„° мӨҖл№„#

мӢңлҰ¬мҰҲм—җм„ң мӮ¬мҡ©н•  мғҳн”ҢмқҖ к°„лӢЁн•ң мқҙм»ӨлЁёмҠӨ лҚ°мқҙн„°лӢӨ. мЈјл¬ё, кі к°қ, мғҒн’Ҳ м„ё н…Ңмқҙлё”. DW лӘЁлҚёл§Ғ 2нҺё м—җм„ң лӢӨлӨҳлҚҳ кө¬мЎ°мҷҖ к°ҷмқҖ лҸ„л©”мқёмқҙлӢӨ.

Bronzeм—җ м Ғмһ¬н–ҲлӢӨ. CSVлҘј мқҪм–ҙм„ң DuckDBм—җ л„Јм—Ҳмқ„ лҝҗ, м–ҙл–Ө ліҖнҷҳлҸ„ н•ҳм§Җ м•Ҡм•ҳлӢӨ. нғҖмһ… мәҗмҠӨнҢ…лҸ„ м•Ҳ н–Ҳкі , м»¬лҹјлӘ…лҸ„ л°”кҫём§Җ м•Ҡм•ҳлӢӨ. мқҙкІҢ Bronzeмқҳ мӣҗм№ҷмқҙлӢӨ.

# Bronze лҚ°мқҙн„° нҷ•мқё

м—¬кё°м„ң л°”лЎң л¶„м„қ мҝјлҰ¬лҘј лҚҳм§Җкі  мӢ¶мқҖ мң нҳ№мқҙ мғқкёҙлӢӨ. м°ём•„м•ј н•ңлӢӨ. Bronze лҚ°мқҙн„°лҘј м§Ғм ‘ л¶„м„қм—җ м“°л©ҙ 3к°ңмӣ” л’Өм—җ лҚ°мқҙн„° лҠӘм—җ л№ м§„лӢӨ. лӢӨмқҢ кёҖм—җм„ң BronzeлҘј SilverлЎң мҳ¬лҰ¬лҠ” кіјм •мқ„ лӢӨлЈ¬лӢӨ.

мҷң мқҙл ҮкІҢк№Ңм§Җ лӮҳлҲ„лҠ”к°Җ#

л Ҳмқҙм–ҙлҘј лӮҳлҲ„л©ҙ лҠҗл Өм§Җм§Җ м•ҠлғҗлҠ” м§Ҳл¬ёмқ„ л°ӣлҠ”лӢӨ. м ҖмһҘ кіөк°„лҸ„ лҚ” м“°кі , ліҖнҷҳ лӢЁкі„лҸ„ лҠҳм–ҙлӮҳлӢҲк№Ң.

л§һлӢӨ. лҢҖмӢ  м„ё к°Җм§ҖлҘј м–»лҠ”лӢӨ.

мһ¬мІҳлҰ¬к°Җ к°ҖлҠҘн•ҳлӢӨ. Silver лЎңм§Ғм—җ лІ„к·ёк°Җ мһҲмңјл©ҙ Bronzeм—җм„ң лӢӨмӢң л§Ңл“Өл©ҙ лҗңлӢӨ. мӣҗліёмқҙ мӮҙм•„мһҲмңјлӢҲк№Ң. Bronze м—Ҷмқҙ Silverл§Ң мһҲмңјл©ҙ мҶҢмҠӨ мӢңмҠӨн…ңм—җм„ң лӢӨмӢң лҒҢм–ҙмҷҖм•ј н•ңлӢӨ.

л¬ём ң м¶”м Ғмқҙ лҗңлӢӨ. Goldмқҳ мҲ«мһҗк°Җ мқҙмғҒн•ҳл©ҙ SilverлҘј ліҙкі , Silverк°Җ мқҙмғҒн•ҳл©ҙ BronzeлҘј ліёлӢӨ. м–ҙлҠҗ л Ҳмқҙм–ҙм—җм„ң л¬ём ңк°Җ мғқкІјлҠ”м§Җ нҠ№м •н•  мҲҳ мһҲлӢӨ.

м—ӯн• мқҙ л¶„лҰ¬лҗңлӢӨ. лҚ°мқҙн„° м—”м§ҖлӢҲм–ҙлҠ” BronzeвҶ’SilverлҘј мұ…мһ„м§Җкі , л¶„м„қ м—”м§ҖлӢҲм–ҙлҠ” SilverвҶ’GoldлҘј мұ…мһ„м§„лӢӨ. м„ңлЎңмқҳ мҳҒм—ӯмқ„ кұҙл“ңлҰ¬м§Җ м•Ҡм•„лҸ„ лҗңлӢӨ.

нҒҙлқјмҡ°л“ң нҷҳкІҪм—җм„ң мҠӨнҶ лҰ¬м§Җ л№„мҡ©мқҖ кұ°мқҳ л¬ҙмӢңн•  мҲҳ мһҲлҠ” мҲҳмӨҖмқҙлӢӨ. л Ҳмқҙм–ҙлҘј н•ҳлӮҳ лҚ” л‘җлҠ” л№„мҡ©ліҙлӢӨ, лҚ°мқҙн„° лҠӘм—җ л№ мЎҢмқ„ л•Ңмқҳ л№„мҡ©мқҙ нӣЁм”¬ нҒ¬лӢӨ.

мӢӨл¬ҙ м°ёкі : Airflow DAG#

мқҙ мӢңлҰ¬мҰҲмқҳ мӢӨмҠөмқҖ Colab + dbtлЎң м§„н–үн•ҳм§Җл§Ң, мӢӨл¬ҙм—җм„ңлҠ” Airflowк°Җ нҢҢмқҙн”„лқјмқёмқ„ мҠӨмјҖмӨ„л§Ғн•ҳкі  мҳӨмјҖмҠӨнҠёл Ҳмқҙм…ҳн•ңлӢӨ. л©”лӢ¬лҰ¬мҳЁ м•„нӮӨн…ҚмІҳмқҳ Bronze вҶ’ Silver вҶ’ Gold нқҗлҰ„мқ„ Airflow DAGмңјлЎң м§ңл©ҙ мқҙлҹ° лӘЁм–‘мқҙлӢӨ.

from airflow import DAG
from airflow.operators.bash import BashOperator
from datetime import datetime

with DAG(
    dag_id='medallion_pipeline',
    schedule='0 6 * * *',  # л§Өмқј мҳӨм „ 6мӢң
    start_date=datetime(2026, 1, 1),
    catchup=False,
) as dag:

    bronze = BashOperator(
        task_id='load_bronze',
        bash_command='python scripts/load_bronze.py',
    )

    silver = BashOperator(
        task_id='run_silver',
        bash_command='cd dbt_project && dbt run --select staging',
    )

    gold = BashOperator(
        task_id='run_gold',
        bash_command='cd dbt_project && dbt run --select marts',
    )

    bronze >> silver >> gold

bronze >> silver >> gold. мқҳмЎҙ кҙҖкі„к°Җ н•ң мӨ„лЎң мқҪнһҢлӢӨ. Bronze м Ғмһ¬к°Җ лҒқлӮҳм•ј Silverк°Җ лҸҢкі , Silverк°Җ лҒқлӮҳм•ј Goldк°Җ лҸҲлӢӨ. Airflowк°Җ мқҙ мҲңм„ңлҘј ліҙмһҘн•ҳкі , мӢӨнҢЁн•ҳл©ҙ м•ҢлҰјмқ„ ліҙлӮҙкі , мһ¬мӢӨн–үлҸ„ мІҳлҰ¬н•ңлӢӨ.

dbtлҠ” “л¬ҙм—Үмқ„ ліҖнҷҳн• м§Җ"лҘј м •мқҳн•ҳкі , AirflowлҠ” “м–ём ң, м–ҙл–Ө мҲңм„ңлЎң мӢӨн–үн• м§Җ"лҘј м •мқҳн•ңлӢӨ. м—ӯн• мқҙ лӢӨлҘҙлӢӨ.

лӢӨмқҢ кёҖм—җм„ңлҠ” Bronze л Ҳмқҙм–ҙлҘј ліёкІ©м ҒмңјлЎң лӢӨлЈ¬лӢӨ. Full LoadмҷҖ Incremental Loadмқҳ м°Ёмқҙ, мҰқл¶„ м Ғмһ¬мқҳ кё°мӨҖ м»¬лҹјмқ„ м–ҙл–»кІҢ мһЎлҠ”м§Җ.
