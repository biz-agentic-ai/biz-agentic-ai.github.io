  3. 3. On-Site GEO кё°мҲ  кө¬мЎ° - мғҒн’Ҳ DBм—җм„ң JSON-LDк№Ңм§Җ вҶҗ нҳ„мһ¬ кёҖ

JSON-LDлҘј м–ҙл””м„ң л§Ңл“Өм–ҙм„ң м–ҙл””м—җ л„ЈлҠҗлғҗ#

мқҙм „ кёҖ м—җм„ң н”Ңлһ«нҸјлі„ мқёмҡ© мҶҢмҠӨк°Җ лӢӨлҘҙлӢӨлҠ” кұё нҷ•мқён–ҲлӢӨ. GeminiлҠ” кіөмӢқ мӮ¬мқҙнҠёлҘј, ChatGPTлҠ” л””л үнҶ лҰ¬лҘј, PerplexityлҠ” м»Өл®ӨлӢҲнӢ°лҘј м„ нҳён•ңлӢӨ. кіөнҶөм җмқҙ н•ҳлӮҳ мһҲлӢӨ. м–ҙл–Ө н”Ңлһ«нҸјмқҙл“  кө¬мЎ°нҷ” лҚ°мқҙн„°к°Җ мһҲлҠ” нҺҳмқҙм§Җмқҳ мқёмҡ© нҷ•лҘ мқҙ лҶ’лӢӨлҠ” кұ°лӢӨ.

к·ёлһҳм„ң On-Site GEOмқҳ кё°мҲ м Ғ н•өмӢ¬мқҖ кІ°көӯ мқҙ м§Ҳл¬ёмңјлЎң мҲҳл ҙн•ңлӢӨ. мғҒн’Ҳ л§ҲмҠӨн„° DBм—җ мһҲлҠ” лҚ°мқҙн„°лҘј м–ҙл–»кІҢ к°Җкіөн•ҙм„ң HTML <head>м—җ JSON-LDлЎң кҪӮмқ„ кІғмқёк°Җ.

лӢЁмҲңн•ҙ ліҙмқҙм§Җл§Ң, мӢӨм ңлЎң мҶҗлҢҖліҙл©ҙ м–ҪнҳҖ мһҲлҠ” кІҢ н•ңл‘ҳмқҙ м•„лӢҲлӢӨ. мғҒн’Ҳ DB н•„л“ңлӘ…мқҖ м•Ҫм–ҙ нҲ¬м„ұмқҙкі , AIк°Җ мқҙн•ҙн•  мҲҳ мһҲлҠ” мҶҚм„ұмқҖ DBм—җ м—Ҷкі , SPAлЎң л§Ңл“Өм–ҙм§„ мӮ¬мқҙнҠёлҠ” нҒ¬лЎӨлҹ¬к°Җ JSON-LDлҘј лӘ» мқҪлҠ”лӢӨ. мқҙ кёҖм—җм„ңлҠ” мқҙ л¬ём ңл“Өмқ„ м–ҙл–Ө кө¬мЎ°лЎң н’Җ мҲҳ мһҲлҠ”м§Җ лӢӨлЈ¬лӢӨ.

GEO мӢңмҠӨн…ңмқҳ лҸҷмӢ¬мӣҗ кө¬мЎ°#

GEO мӢңмҠӨн…ңмқҖ м•ҲмӘҪм—җм„ң л°”к№ҘмңјлЎң нҷ•мһҘлҗҳлҠ” 4к°ң л Ҳмқҙм–ҙлЎң мқҙлЈЁм–ҙм§„лӢӨ.

л Ҳмқҙм–ҙкө¬м„ұмҡ”мҶҢм—ӯн• 
CoreмғҒн’Ҳ л§ҲмҠӨн„° DBSSOT(Single Source of Truth). лӘЁл“  лҚ°мқҙн„°мқҳ мӣҗмІң
Channelмӣ№мӮ¬мқҙнҠё / лӘЁл°”мқј м•ұJSON-LD мӮҪмһ…, SSR л ҢлҚ”л§Ғ
APIмғҒн’Ҳ мЎ°нҡҢ APIAI м—җмқҙм „нҠёк°Җ нҳём¶ңн•  мҲҳ мһҲлҠ” мқён„°нҺҳмқҙмҠӨ
AgentChatGPT / Gemini / Perplexityмөңмў… мҶҢл№„мһҗ м ‘м җ

Coreм—җм„ң мӢңмһ‘н•ҙм„ң Channelмқ„ кұ°міҗ Agentк№Ңм§Җ лҚ°мқҙн„°к°Җ нқҳлҹ¬к°„лӢӨ. к°Ғ л Ҳмқҙм–ҙлҘј м§ҖлӮ  л•Ңл§ҲлӢӨ лҚ°мқҙн„°мқҳ нҳ•нғңк°Җ л°”лҖҗлӢӨ. DBмқҳ raw н•„л“ңк°Җ кө¬мЎ°нҷ”лҗң JSON-LDк°Җ лҗҳкі , к·ёкІҢ AI лӢөліҖмқҳ мқёмҡ© м¶ңмІҳк°Җ лҗңлӢӨ.

м—¬кё°м„ң лҶ“м№ҳкё° мү¬мҡҙ кІҢ API л Ҳмқҙм–ҙлӢӨ. JSON-LDл§Ң мһҳ л„Јмңјл©ҙ лҗҳлҠ” кұ° м•„лӢҢк°Җ мӢ¶м§Җл§Ң, ChatGPT PluginsмқҙлӮҳ MCP(Model Context Protocol) к°ҷмқҖ AI м—җмқҙм „нҠё м—°лҸҷк№Ңм§Җ кі л Өн•ҳл©ҙ лі„лҸ„мқҳ API л Ҳмқҙм–ҙк°Җ н•„мҡ”н•ҳлӢӨ. м§ҖкёҲ лӢ№мһҘмқҖ м•„лӢҲлҚ”лқјлҸ„ м„Өкі„ лӢЁкі„м—җм„ң кі л Өн•ҙл‘җл©ҙ лӮҳмӨ‘м—җ лҚң кі мғқн•ңлӢӨ.

лҚ°мқҙн„° нҢҢмқҙн”„лқјмқё 3лӢЁкі„#

мғҒн’Ҳ descriptionмқ„ нҶөм§ёлЎң кҙҖлҰ¬н•ҳм§Җ м•Ҡкі , н•„л“ң лӢЁмң„лЎң л¶„н•ҙн•ҳл©ҙ AIк°Җ м •нҷ•нһҲ мқёмҡ©н•ңлӢӨ. мқҙкІҢ нҢҢмқҙн”„лқјмқёмқҳ н•өмӢ¬ м•„мқҙл””м–ҙлӢӨ.

1лӢЁкі„: DB м •м ң - кё°мЎҙ н•„л“ң л§Өн•‘#

кё°мЎҙ мғҒн’Ҳ л§ҲмҠӨн„° DBм—җм„ң Schema.org н•„л“ңлЎң л§Өн•‘н•ҳлҠ” лӢЁкі„лӢӨ. мғҲ лҚ°мқҙн„°лҘј л§Ңл“ңлҠ” кІҢ м•„лӢҲлқј мһҲлҠ” лҚ°мқҙн„°лҘј м •лҰ¬н•ҳлҠ” кұ°лӢӨ.

GS1 н‘ңмӨҖ мӢқлі„мһҗмқёлҚ°, к°ҷмқҖ мғҒн’ҲмқҙлқјлҸ„ мҡ©лҹүмқҙлӮҳ л§ӣмқҙ лӢӨлҘҙл©ҙ GTINмқҙ лӢ¬лқјм•ј н•ңлӢӨ. “мҙҲмҪ”мҠӨнӢұ мҳӨлҰ¬м§Җл„җ"кіј “мҙҲмҪ”мҠӨнӢұ м•„лӘ¬л“ң"лҘј н•ҳлӮҳмқҳ лҢҖн‘ңмҪ”л“ңлЎң л¬¶м–ҙлҶ“мңјл©ҙ AIлҠ” л‘ҳмқ„ кө¬л¶„н•ҳм§Җ лӘ»н•ңлӢӨ.

2лӢЁкі„: LLM м¶”м¶ң - AI кё°л°ҳ мҶҚм„ұ мһҗлҸҷ мғқм„ұ#

нҢҢмқҙн”„лқјмқём—җм„ң к°ҖмһҘ мӮ¬лһҢ мҶҗмқҙ л§Һмқҙ нғҖлҠ” л¶Җл¶„мқҙ мғҒн’Ҳ м„ӨлӘ…(description)мқҙлӢӨ. AIк°Җ мқёмҡ©н•ҳкё° мўӢмқҖ descriptionм—җлҠ” нҢЁн„ҙмқҙ мһҲлӢӨ.

мӮ¬мӢӨ кё°л°ҳ - к°қкҙҖм Ғ м •ліҙл§Ң л„ЈлҠ”лӢӨ. “м—…кі„ мөңкі "лӮҳ “кі к°қ л§ҢмЎұ 1мң„” к°ҷмқҖ кҙ‘кі  л¬ёкө¬лҠ” AIк°Җ л¬ҙмӢңн•ңлӢӨ.

SSRмқҙ н•„мҲҳмқё мқҙмң #

JSON-LDлҘј л§Ңл“Өм—Ҳм–ҙлҸ„ AI нҒ¬лЎӨлҹ¬к°Җ лӘ» мқҪмңјл©ҙ мҶҢмҡ©м—ҶлӢӨ. м—¬кё°м„ң SPA(Single Page Application)к°Җ л°ңлӘ©мқ„ мһЎлҠ”лӢӨ.

SPAлҠ” лёҢлқјмҡ°м Җм—җм„ң JavaScriptлҘј мӢӨн–үн•ҙм•ј мҪҳн…җмё к°Җ л ҢлҚ”л§ҒлҗңлӢӨ. мӮ¬лһҢ лҲҲм—җлҠ” м •мғҒмңјлЎң ліҙмқҙм§Җл§Ң, GPTBotмқҙлӮҳ Google-Extended к°ҷмқҖ AI нҒ¬лЎӨлҹ¬лҠ” лҢҖл¶Җл¶„ JSлҘј мӢӨн–үн•ҳм§Җ м•ҠлҠ”лӢӨ. <head>м—җ JSON-LDлҘј л„Јм—Ҳм–ҙлҸ„ м„ңлІ„к°Җ л№Ҳ HTMLмқ„ ліҙлӮҙл©ҙ нҒ¬лЎӨлҹ¬ мһ…мһҘм—җм„ңлҠ” м—ҶлҠ” кұ°лӮҳ л§Ҳм°¬к°Җм§ҖлӢӨ.

SSR(Server-Side Rendering)лЎң м „нҷҳн•ҳл©ҙ м„ңлІ„м—җм„ң мҷ„м„ұлҗң HTMLмқ„ ліҙлӮҙкё° л•Ңл¬ём—җ нҒ¬лЎӨлҹ¬к°Җ JS мӢӨн–ү м—Ҷмқҙ JSON-LDлҘј л°”лЎң мқҪлҠ”лӢӨ.

Next.js App Router кё°мӨҖмңјлЎң ліҙл©ҙ мқҙл ҮлӢӨ:

// app/product/[id]/page.tsx
export default async function ProductPage({ params }) {
  const product = await fetchProduct(params.id);

  const jsonLd = {
    "@context": "https://schema.org",
    "@type": "Product",
    "name": product.name,
    "gtin13": product.gtin,
    "brand": { "@type": "Brand", "name": product.brand },
    "description": product.description,
    "image": product.imageUrl,
    "offers": {
      "@type": "Offer",
      "price": product.price,
      "priceCurrency": "KRW",
      "availability": "https://schema.org/InStock",
      "url": product.pageUrl
    }
  };

  return (
    <>
      <script
        type="application/ld+json"
        dangerouslySetInnerHTML={{
          __html: JSON.stringify(jsonLd)
        }}
      />
      <ProductDetail product={product} />
    </>
  );
}

м„ңлІ„м—җм„ң fetchProductлЎң DB лҚ°мқҙн„°лҘј к°Җм ёмҳӨкі , JSON-LD к°қмІҙлҘј л§Ңл“Өм–ҙм„ң <script> нғңк·ёлЎң мЈјмһ…н•ңлӢӨ. мқҙ HTMLмқҙ нҒ¬лЎӨлҹ¬м—җкІҢ к·ёлҢҖлЎң м „лӢ¬лҗңлӢӨ.

SSR лҸ„мһ…мқҙ л¶ҖлӢҙмҠӨлҹ¬мҡҙ кІҪмҡ°, кіјлҸ„кё°м ҒмңјлЎң Google Tag Manager(GTM)лҘј нҷңмҡ©н•ҙ JSON-LDлҘј мЈјмһ…н•ҳлҠ” л°©лІ•лҸ„ мһҲлӢӨ. мҷ„м „н•ң SSRліҙлӢӨлҠ” нҡЁкіјк°Җ л–Ём–ҙм§Җм§Җл§Ң SPAлҘј лӢ№мһҘ м „нҷҳн•  мҲҳ м—Ҷмқ„ л•Ң м“ё мҲҳ мһҲлҠ” мҡ°нҡҢ л°©м•ҲмқҙлӢӨ.

SSR м Ғмҡ© мӢң мЈјмқҳм җ#

н•ӯлӘ©мһҘм җлӢЁм җк·№ліө л°©м•Ҳ
SEO мөңм Ғнҷ”нҒ¬лЎӨлҹ¬к°Җ JS м—Ҷмқҙ мҰүмӢң мқёмӢқмҙҲкё° к°ңл°ң л№„мҡ©SDK/кіөнҶө лӘЁл“Ҳ м ңкіө
лҚ°мқҙн„° л°ҳмҳҒDB ліҖкІҪ мӢң мһҗлҸҷ м—…лҚ°мқҙнҠём„ңлІ„ л¶Җн•ҳ мҰқк°ҖRedis мәҗмӢұ + ISR нҷңмҡ©
мӨ‘м•ҷ кҙҖлҰ¬м „мІҙ мӮ¬мқҙнҠё мқјкҙ„ м Ғмҡ©к°ңл°ңнҢҖ мқҳмЎҙлҸ„кҙҖлҰ¬ мҪҳмҶ”м—җм„ң л№„к°ңл°ңмһҗ кҙҖлҰ¬
кІҖмҰқ мһҗлҸҷнҷ”л№Ңл“ң мӢң мң нҡЁм„ұ кІҖмҰқ нҸ¬н•Ёл Ҳкұ°мӢң мӢңмҠӨн…ң м „нҷҳGTM н•ҳмқҙлёҢлҰ¬л“ң лі‘н–ү

м„ңлІ„ л¶Җн•ҳлҠ” Redis мәҗмӢұкіј ISR(Incremental Static Regeneration)лЎң мғҒлӢ№ л¶Җл¶„ н•ҙмҶҢлҗңлӢӨ. мғҒн’Ҳ м •ліҙк°Җ л°”лҖҢм§Җ м•ҠлҠ” н•ң мәҗмӢңлҗң HTMLмқ„ к·ёлҢҖлЎң м“°л©ҙ лҗңлӢӨ.

лҚ°мқҙн„° мӢ м„ лҸ„к°Җ мқёмҡ©мқ„ мўҢмҡ°н•ңлӢӨ#

кө¬мЎ°нҷ”лҘј мһҳ н•ҙлҶ”лҸ„ мҳӨлһҳлҗң лҚ°мқҙн„°лҠ” л°ҖлҰ°лӢӨ.

Perplexityм—җм„ң лҶ’мқҖ мқёмҡ©мқ„ л°ӣмқҖ нҺҳмқҙм§ҖлҘј л¶„м„қн•ҙліҙл©ҙ, 4л¶„мқҳ 3 мқҙмғҒмқҙ н•ң лӢ¬ мқҙлӮҙм—җ м—…лҚ°мқҙнҠёлҗң кІғмқҙм—ҲлӢӨ. ChatGPT ShoppingмқҖ н”јл“ңлҘј 15л¶„ к°„кІ©мңјлЎң к°ұмӢ н•ңлӢӨ (OpenAI). м„қ лӢ¬ л„ҳкІҢ мҶҗ м•Ҳ лҢ„ нҺҳмқҙм§ҖлҠ” AI мқёмҡ© мҲңмң„м—җм„ң л°Җл ӨлӮ  к°ҖлҠҘм„ұмқҙ лҶ’лӢӨ.

мӢ м„ лҸ„ кҙҖлҰ¬ кё°мӨҖмқ„ мһЎмңјл©ҙ мқҙл ҮлӢӨ:

  • н•өмӢ¬ лҚ°мқҙн„° (к°ҖкІ©, мһ¬кі , н”„лЎңлӘЁм…ҳ): 24мӢңк°„ мқҙлӮҙ к°ұмӢ 
  • мқјл°ҳ лҚ°мқҙн„° (мғҒн’Ҳ м„ӨлӘ…, мқҙлҜём§Җ): 7мқј мқҙлӮҙ к°ұмӢ 
  • м •м Ғ лҚ°мқҙн„° (лёҢлһңл“ң м •ліҙ, нҡҢмӮ¬ мҶҢк°ң): мӣ” 1нҡҢ м җкІҖ

sitemap.xmlм—җм„ң lastmod лӮ м§ңлҘј мӢӨм ң м—…лҚ°мқҙнҠё мӢңм җм—җ л§һм¶° к°ұмӢ н•ҳкі , IndexNow APIлЎң ліҖкІҪ мӮ¬н•ӯмқ„ кІҖмғүм—”м§„м—җ мҰүмӢң м•Ңл ӨмЈјлҠ” кІғлҸ„ нҡЁкіјк°Җ мһҲлӢӨ.

кІҖмҰқ - л„Јм—Ҳмңјл©ҙ нҷ•мқён•ҙм•ј н•ңлӢӨ#

JSON-LDлҘј л„Јм—ҲлӢӨкі  лҒқмқҙ м•„лӢҲлӢӨ. мӢӨм ңлЎң нҒ¬лЎӨлҹ¬к°Җ мһҳ мқҪлҠ”м§Җ нҷ•мқён•ҙм•ј н•ңлӢӨ.

Google Rich Results Test - search.google.com/test/rich-results м—җм„ң URLмқ„ мһ…л Ҙн•ҳл©ҙ кө¬мЎ°нҷ” лҚ°мқҙн„°к°Җ м •мғҒ мқёмӢқлҗҳлҠ”м§Җ л°”лЎң нҷ•мқён•  мҲҳ мһҲлӢӨ.

curlлЎң нҒ¬лЎӨлҹ¬ мӢңл®¬л Ҳмқҙм…ҳ - AI нҒ¬лЎӨлҹ¬мқҳ User-AgentлЎң м§Ғм ‘ мҡ”мІӯмқ„ ліҙлӮҙм„ң JSON-LDк°Җ HTMLм—җ нҸ¬н•Ёлҗҳм–ҙ мҳӨлҠ”м§Җ нҷ•мқён•ңлӢӨ.

# GPTBotмңјлЎң мҡ”мІӯ
curl -A "GPTBot" https://www.example.com/product/12345 | grep "application/ld+json"

# HTML мҶҢмҠӨм—җм„ң JSON-LD м¶”м¶ң
curl -s https://www.example.com/product/12345 \
  | grep -oP '<script type="application/ld\+json">.*?</script>'

SPAмқёлҚ° SSR м „нҷҳмқҙ м•Ҳ лҗң мғҒнғңлқјл©ҙ, curl кІ°кіјм—җ JSON-LDк°Җ м•Ҳ лӮҳмҳ¬ мҲҳ мһҲлӢӨ. мқҙкІҢ л°”лЎң SSRмқҙ н•„мҲҳмқё мқҙмң лӢӨ.

нҸјм—җ мғҒн’Ҳ м •ліҙлҘј мһ…л Ҙн•ҳл©ҙ JSON-LDк°Җ мһҗлҸҷ мғқм„ұлҗҳлҠ” л№ҢлҚ”лҸ„ л§Ңл“Өм–ҙл‘җм—ҲлӢӨ. кө¬мЎ°лҘј м§Ғм ‘ л§Ңм ёліҙл©ҙ к°җмқҙ мһЎнһҢлӢӨ.
