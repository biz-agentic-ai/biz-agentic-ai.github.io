“ВіцьЃђВіцьѓцвДѕ ВЋѕ ьЋ┤вЈё вљўвѓўВџћ?”#

ьЂ┤вЮ╝Вџ░вЊю DW ВаёьЎў ьћёвАюВаЮьіИвЦ╝ ьЋўвЕ┤ Ж╝Г вѓўВўцвіћ ВДѕвгИВЮ┤вІц.

Вўеьћёваѕв»ИВіцВЌљВёю ВѕўвЁёЖ░ё Вџ┤ВўЂьЋўвЇў DWвЦ╝ BigQueryвѓў Azure SynapseвАю Вў«ЖИ░віћ Въљвдг. вѕёЖх░Ж░ђЖ░ђ вг╗віћвІц. “Ж▒░ЖИ░віћ Columnar Storage(В╗гвЪ╝ЖИ░в░ў ВађВъЦВєї)вЮ╝ ВА░ВЮИ в╣ёВџЕВЮ┤ вІцвЦ┤вІцвіћвЇ░, ЖиИвЪгвЕ┤ ВіцьЃђВіцьѓцвДѕ ВЋѕ ьЋ┤вЈё вљўвіћ Ж▒░ ВЋёвІїЖ░ђВџћ?”

вІхВЮё ве╝Вађ вДљьЋўвЕ┤ - ВЃЂьЎЕВЌљ вћ░вЮ╝ вІцвЦ┤вІц. ЖиИвЪ░вЇ░ ВЮ┤ “ВЃЂьЎЕВЌљ вћ░вЮ╝ вІцвЦ┤вІц"Ж░ђ ЖхгВ▓┤ВаЂВю╝вАю вГў вћ░ВаИВЋ╝ ьЋўвіћ Ж▒┤ВДђвЦ╝ ВЋёвіћ ВѓгвъїВЮђ вДјВДђ ВЋівІц.

Вўеьћёваѕв»ИВіц ВІюВаѕВЮў Ж│хВІЮ#

KimballВЮў Dimensional(Star-Schema) вфевЇИвДЂВЮ┤ ВѓгВІцВЃЂ ьЉюВцђВЮ┤вЇў ВІюВаѕВЮ┤ ВъѕВЌѕвІц.

ВЮ┤Вюавіћ вІеВѕюьќѕвІц. вћћВіцьЂг I/OЖ░ђ в╣ёВїїЖ│а, ВА░ВЮИВЮђ вЇћ в╣ёВїївІц. Row ЖИ░в░ў ВіцьєавдгВДђВЌљВёю ьЁїВЮ┤вИћ 10Ж░ювЦ╝ ВА░ВЮИьЋўвЕ┤ В┐╝вдг ВЮЉвІхВЮ┤ вХё вІеВюёвАю вёўВќ┤Ж░ћвІц. ЖиИвъўВёю в»Ивдг ВА░ВЮИьЋ┤ вєЊвіћ Ж▓ї ьЋЕвдгВаЂВЮ┤ВЌѕвІц.

вфевЇИвДЂ ВЮўВѓгЖ▓░ВаЋВЮ┤ Ж│Д Вё▒віЦ ВЮўВѓгЖ▓░ВаЋВЮ┤ВЌѕвІц. Вќ┤вћћЖ╣їВДђ в╣ёВаЋЖиюьЎћьЋа Ж▓ЃВЮИЖ░ђ, ВДЉЖ│ё ьЁїВЮ┤вИћВЮё вфЄ вІеЖ│ёвАю ВїЊВЮё Ж▓ЃВЮИЖ░ђ, ьїїьІ░ВЁў ьѓцвЦ╝ вГўвАю ВъАВЮё Ж▓ЃВЮИЖ░ђ. ВЮ┤ Ж▓░ВаЋвЊцВЮ┤ В┐╝вдг ВЮЉвІх ВІюЖ░ёВЮё В┤ѕ вІеВюёВЌљВёю вХё вІеВюёвАю Ж░ѕвъљвІц.

KimballВЮў в░Ев▓ЋвАаВЮђ ВЮ┤ ВаюВЋй ВЋѕВЌљВёю в╣ёВдѕвІѕВіц Ж░ђвЈЁВё▒Ж│╝ В┐╝вдг Вё▒віЦВЮё вЈЎВІюВЌљ ВъАВю╝вацвіћ ВІювЈёВўђвІц. вХђВёювДѕвІц ВаюЖ░ЂЖ░ЂВЮ┤вЇў В░еВЏљ ьЁїВЮ┤вИћВЮё ВаёВѓг Ж│хьєх ЖИ░ВцђВю╝вАю ьєхВЮ╝(Conformed Dimension)ьЋ┤Вёю ВЮ╝Ж┤ђВё▒ВЮё в│┤ВъЦьЋўЖ│а, Bus MatrixвАю ВаёВѓг ьєхьЋЕВЮё ВёцЖ│ёьЋўЖ│а. в░Ев▓ЋвАа ВъљВ▓┤ВЮў ВЎёВё▒вЈёвіћ ВДђЖИѕ в┤љвЈё вєњвІц.

вгИВаювіћ ВЮ┤ в░Ев▓ЋвАаВЮ┤ вДївЊцВќ┤ВДё ВаёВаюЖ░ђ в░ћвђїВЌѕвІцвіћ Ж▒░вІц.

ьЂ┤вЮ╝Вџ░вЊюЖ░ђ в░ћЖЙ╝ ВаёВаювЊц#

ьЂ┤вЮ╝Вџ░вЊю DWвАю ВўцвЕ┤Вёю вг╝вдгВаЂ ВаюВЋйВЮ┤ Жи╝в│ИВаЂВю╝вАю вІгвЮ╝ВАївІц.

Columnar Storage. BigQuery, Redshift, Synapse вфевЉљ В╗гвЪ╝ ЖИ░в░ўВЮ┤вІц. SELECTВЌљВёю ьЋёВџћьЋю В╗гвЪ╝вДї ВЮйвіћвІц. В╗гвЪ╝ВЮ┤ Вѕўв░▒ Ж░юВЮИ ьЁїВЮ┤вИћВЌљВёю Вёювёѕ Ж░ювДї ВА░ьџїьЋўвЕ┤ ЖиИЖ▓ЃвДї ВіцВ║ћьЋювІц. Row ЖИ░в░ўВЌљВёювіћ ВаёвХђ ВЮйВќ┤ВЋ╝ ьќѕвІц.

Compute/Storage вХёвдг. ВіцьєавдгВДђЖ░ђ ВІИВАївІц. ВцЉв│х ВађВъЦьЋ┤вЈё в╣ёВџЕ вХђвІ┤ВЮ┤ ВъЉвІц. Вўеьћёваѕв»ИВіцВЌљВёю вћћВіцьЂг ВџЕвЪЅ ВЋёвЂ╝вацЖ│а ВаЋЖиюьЎћвЦ╝ Ж│ав»╝ьЋўвЇў Ж▓ЃЖ│╝віћ ВЃЂьЎЕВЮ┤ вІцвЦ┤вІц.

MPP ВЋёьѓцьЁЇВ▓ў. вїђЖиювфе в│Љваг В▓ўвдгЖ░ђ ЖИ░в│ИВЮ┤вІц. ВА░ВЮИ в╣ёВџЕВЮ┤ Вўеьћёваѕв»ИВіц RDBMS вїђв╣ё ВЃЂвїђВаЂВю╝вАю вѓ«ВЋёВАївІц. вг╝вАа Ж│хВДювіћ ВЋёвІѕвІц - ВЁћьћїВЮ┤ в░юВЃЮьЋўвЕ┤ ВЌгВаёьъѕ віљвдгвІц. ьЋўВДђвДї “ВА░ВЮИВЮђ вг┤ВА░Ж▒┤ ьћ╝ьЋўвЮ╝"віћ ВЏљВ╣ЎВЮ┤ вЇћ ВЮ┤ВЃЂ ВаѕвїђВаЂВЮ┤ВДђ ВЋівІц.

ELT ьїевЪгвІцВъё. ВЏљв│ИВЮё ве╝Вађ ВаЂВъгьЋўЖ│а, DW ВЋѕВЌљВёю в│ђьЎўьЋювІц. в│ђьЎў вАюВДЂВЮё DW ВЌћВДёВЮў В╗┤ьЊеьїЁ ьїїВЏївАю В▓ўвдгьЋювІц. ETL ВІюВаѕВ▓ўвЪ╝ в│ђьЎў Вёюв▓ёвЦ╝ в│ёвЈёвАю вЉљЖ│а “вІц вДївЊцВќ┤Вёю вёБвіћ” ЖхгВА░Ж░ђ ВЋёвІѕвІц.

в░ўВаЋьўЋ вЇ░ВЮ┤ьё░ ВДђВЏљ. JSON, ARRAY, STRUCTвЦ╝ вёцВЮ┤ьІ░вИївАю вІцвБгвІц. ВаёьєхВаЂВЮИ Ж┤ђЖ│ёьўЋ вфевЇИвАю ьњђЖИ░ Вќ┤вацВЏавЇў ВюаВЌ░ьЋю ВіцьѓцвДѕвЦ╝ DW ВЋѕВЌљВёю ВДЂВаЉ В▓ўвдгьЋа Вѕў ВъѕвІц.

ВЮ┤ в│ђьЎћвЊцВЮ┤ ЖИ░ВА┤ вфевЇИвДЂ ВЏљВ╣ЎВЮў Жи╝Ж▒░вЦ╝ ьЮћвЊавІц. ьЋўВДђвДї Жи╝Ж▒░Ж░ђ ВЋйьЋ┤ВДё Ж▓ЃЖ│╝ ВЏљВ╣ЎВЮ┤ ьІђвд░ Ж▓ЃВЮђ вІцвЦИ ВЮ┤ВЋ╝ЖИ░вІц.

ВёИ Ж░ђВДђ ВёаьЃЮВДђ#

ьЂ┤вЮ╝Вџ░вЊю DWВЌљВёю ВъљВБ╝ в╣ёЖхљвљўвіћ ВёИ Ж░ђВДђ ВаЉЖи╝в▓ЋВЮ┤ ВъѕвІц.

1. Kimball Dimensional вфевЇИвДЂ#

ВіцьЃђВіцьѓцвДѕ, ьїЕьіИВЎђ вћћвЕўВЁў, Conformed Dimension. ВЌгВаёьъѕ Ж░ђВъЦ вёљвдг ВЊ░ВЮИвІц.

ьЂ┤вЮ╝Вџ░вЊюВЌљВёювЈё ВюаьџеьЋю ВЮ┤ВюаЖ░ђ ВъѕвІц. в╣ёВдѕвІѕВіц ВѓгВџЕВъљЖ░ђ ВЮ┤ьЋ┤ьЋўЖИ░ ВЅйвІц. “вДцВХю ьїЕьіИвЦ╝ Ж│аЖ░Ю вћћвЕўВЁўВю╝вАю ВъўвЮ╝в│ИвІц"віћ ЖхгВА░віћ BI вЈёЖхгВЎђвЈё ЖХЂьЋЕВЮ┤ ВбІвІц. Power BI, Tableau, Looker ВаёвХђ ВЮ┤ ЖхгВА░вЦ╝ ВаёВаювАю ВхюВаЂьЎћвљўВќ┤ ВъѕвІц.

вІгвЮ╝ВДё Ж▓ЃвЈё ВъѕвІц. ВДЉЖ│ё ьЁїВЮ┤вИћВЮё в»Ивдг ВїЊВЋёвЉў ьЋёВџћЖ░ђ ВцёВЌѕвІц. В╗гвЪ╝вѓў ВіцьєавдгВДђВЌљВёю ВЏљв│И ьїЕьіИвЦ╝ в░ћвАю ВДЉЖ│ёьЋ┤вЈё ВХЕвХёьъѕ в╣авЦ┤вІц. SCD Type 2 Ж░ЎВЮђ ВЮ┤ваЦ Ж┤ђвдгвЈё ВіцьєавдгВДђ вХђвІ┤ВЮ┤ ВъЉВЋёВаИВёю вЇћ ВаЂЖи╣ВаЂВю╝вАю ВЊИ Вѕў ВъѕЖ▓ї вљљвІц.

ВЋйВаљВЮђ ВюаВЌ░Вё▒ВЮ┤вІц. ВіцьѓцвДѕЖ░ђ в░ћвђївЕ┤ ьїЕьіИ/вћћвЕўВЁў ЖхгВА░вЦ╝ ВъгВёцЖ│ёьЋ┤ВЋ╝ ьЋювІц. ВЋаВъљВЮ╝ьЋўЖ▓ї в╣авЦ┤Ж▓ї в░ћвђївіћ ВџћЖхгВѓгьЋГВЌљ вїђВЮЉьЋўЖИ░Ж░ђ ЖхгВА░ВаЂВю╝вАю Вќ┤вахвІц.

2. Data Vault 2.0#

Hub(в╣ёВдѕвІѕВіц ьѓц), Satellite(ВєЇВё▒), Link(Ж┤ђЖ│ё). ВЏљв│И вЇ░ВЮ┤ьё░вЦ╝ Въѕвіћ ЖиИвїђвАю ВЮ┤ваЦЖ│╝ ьЋеЖ╗ў ВађВъЦьЋўвіћ вЇ░ В┤ѕВаљВЮё вДъВХў в░Ев▓ЋвАаВЮ┤вІц.

Ж░ЋВаљВЮђ вфЁьЎЋьЋўвІц. Ж░љВѓг ВХћВаЂВё▒(auditability). ВЏљв│ИВЮ┤ ВќИВаю Вќ┤вќц Ж░њВЮ┤ВЌѕвіћВДђвЦ╝ ВЎёВаёьЋўЖ▓ї в│┤ВА┤ьЋювІц. ВєїВіц ВІюВіцьЁюВЮ┤ ВХћЖ░ђвљўЖ▒░вѓў ВіцьѓцвДѕЖ░ђ в│ђЖ▓йвљўВќ┤вЈё ЖИ░ВА┤ ЖхгВА░ВЌљ ВўЂьќЦВЮё ВБ╝ВДђ ВЋівіћвІц. в│Љваг ВаЂВъгЖ░ђ Ж░ђвіЦьЋ┤Вёю ELT ьїевЪгвІцВъёЖ│╝вЈё Въў вДъвіћвІц.

ьўёВІцВаЂВЮИ Ж▒Ивд╝вЈїВЮ┤ ВъѕвІц. ВДЂВаЉ В┐╝вдгьЋўЖИ░Ж░ђ Ж╣ївІцвАГвІц. Hub-Satellite ВА░ВЮИВЮё ВЌгвЪг в▓ѕ Ж▒░В│љВЋ╝ ьЋўвѓўВЮў в╣ёВдѕвІѕВіц ВЌћьё░ьІ░Ж░ђ вѓўВўевІц. ЖиИвъўВёю Ж▓░ЖхГ ьћёваѕВааьЁїВЮ┤ВЁў ваѕВЮ┤Вќ┤(в│┤ьєх ВіцьЃђВіцьѓцвДѕ)вЦ╝ вћ░вАю вДївЊцВќ┤ВЋ╝ ьЋювІц. вфевЇИвДЂ ваѕВЮ┤Вќ┤Ж░ђ ьЋўвѓў вЇћ ВЃЮЖИ░віћ ВЁѕВЮ┤вІц. ьїђВЮ┤ Data Vault Ж▓йьЌўВЮ┤ ВЌєВю╝вЕ┤ ьЋЎВіх Ж│АВёавЈё Ж░ђьїївЦ┤вІц.

ЖИѕВюх, ВЮўвБї Ж░ЎВЮђ ЖиюВаю Вѓ░ВЌЁВЌљВёю Ж░љВѓг ВХћВаЂВЮ┤ ьЋёВѕўВЮ╝ вЋї, вўљвіћ ВєїВіц ВІюВіцьЁюВЮ┤ ВѕўВІювАю ВХћЖ░ђвљўвіћ ьЎўЖ▓йВЌљВёю Ж░ЋьЋўвІц.

3. One Big Table (OBT)#
